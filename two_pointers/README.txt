# Two Pointers Technique (Java)

This folder contains problems solved using the Two Pointers pattern.

The Two Pointers technique is commonly used to optimize solutions
that involve array traversal, searching, reversing, partitioning,
and pair-based problems.

---

## What You Will Learn

- Opposite direction pointers
- Same direction pointers
- In-place array modifications
- Pair-based problem solving
- Efficient traversal strategies

---

## Problems Included

- ReverseArrayTP.java  
- MoveZeros.java  
- MoveNegativeTP.java  
- TwoSumSorted.java  
- TwoSumBubbleSort.java  
- PairWithDifference.java  
- PalindromeTP.java  
- StringPalindromeTP.java  
- MergeSortedArrays.java  

---

## Key Concepts Practiced

- Pointer movement logic
- Avoiding nested loops
- Space optimization
- Time complexity improvements
